# 接口：login  [返回](../README.md)
用例： [登录](../user/登陆用例.md)

- 功能：
    登录到实验管理平台。

- 权限：
    访客。

- API请求地址：
   http://localhost/yuhang/api/login

- 请求方式 ：
    POST

- 请求实例：

        {
            "id":"201510414101",
            "password":"123456",
            "type":"学生"
        }

- 请求参数说明:

  |参数名称|说明|
  |:---------:|:--------------------------------------------------------|
  |id|学生学号或者老师的工号|
  |password|用户的密码|
  |type|用户类型，学生或者老师或者管理员|

- 返回实例：

        {
            "status": true,
            "info": null,
        }

- 返回参数说明：

  |参数名称|说明|
  |:---------:|:--------------------------------------------------------|
  |status|bool类型，true表示正确的返回，false表示有错误|
  |info|返回结果说明信息|
