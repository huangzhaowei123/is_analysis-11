# 接口：logout  [返回](../README.md)
用例： [登出](../user/登出用例.md)

- 功能：
    已经登录的用户登出平台。

- 权限：
    学生/老师/管理员已经登录。

- API请求地址：
    http://localhost/yuhang/api/logout

- 请求方式 ：
    POST

- 请求实例：

        {
            "user_id":"201510414101"
        }

- 请求参数说明:

  |参数名称|说明|
  |:---------:|:--------------------------------------------------------|
  |user_id|已经登录用户的用户账号|

- 返回实例：

        {
            "status": true,
            "info": null,

        }

- 返回参数说明：

  |参数名称|说明|
  |:---------:|:--------------------------------------------------------|
  |status|bool类型，true表示正确的返回，false表示有错误|
  |info|返回结果说明信息|
